== Description ==

ADN for WP is a lightweight WordPress plugin that allows you to add an App.net share (Post) button at the bottom of posts/pages or posts only.

== Installation ==

1. Upload `adnwp.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Who do I talk to if I have a feature request or support issue? =

ADN for WP is made by the folks at Virtuous Giant. You can learn more about them here: http://virtuousgiant.com

If you need to get in touch, send an email to hello@virtuousgiant.com.

= What is App.net? =

App.net is a subscription based micro-blogging platform, like Twitter, with messages of 140 characters or less.

= Why App.net? =

There are plenty of ways to share posts via Twitter, Facebook, and other social networks, but few for App.net (ADN). We wanted to change that.

= Is this plugin compatible with other social sharing plugins? =

It should be, yes. The plugin framework is very light weight and should play nice with other plugins. If you find evidence to the contrary, please contact us using the information above.
